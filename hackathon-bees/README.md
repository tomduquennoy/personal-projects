# 🐝 Andrena Bee Species Classification

Kaggle competition: [Reconnaissance d'abeilles françaises](https://www.kaggle.com/competitions/reconnaissance-dabeilles-francaises/overview)  
Organised by **ISAE-SUPAERO** in collaboration with **SETE-CNRS**


## Context

Pollinators are essential to ecosystems and are a focus of biodiversity monitoring efforts.
The [SETE-CNRS](https://www.sete.fr/) lab is developing an **Entomoscope**: a non-invasive camera device placed above flowers to automatically capture bee visits.

This hackathon provides a species-annotated image database of **Andrena** bees (the most represented genus in the bee family) and challenges teams to build a model that identifies the Andrena species in each photo.
